HOW TO USE
================================================================================

QUICK START:

1. Review the template and generator:
   cat php-site/deploy/templates/location-service-scalable.php
   cat php-site/deploy/scripts/generate-location-service-pages.php

2. Test single page generation:
   php php-site/deploy/scripts/generate-location-service-pages.php \
       --service=seo --location=edmonton --dry-run --verbose

3. Generate actual page:
   php php-site/deploy/scripts/generate-location-service-pages.php \
       --service=seo --location=edmonton

4. Verify output:
   cat public/services/seo-in-edmonton/index.php | head -50

   php php-site/deploy/scripts/generate-location-service-pages.php --verbose
