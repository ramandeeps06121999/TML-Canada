## 7. Recommended Keyword Research Tools & Process

### Tools
- **DataForSEO / Ahrefs / SEMrush** - Volume, KD, CPC data for Canadian market
- **Google Keyword Planner** - CPC and competition data (verify commercial intent)
- **Google Search Console** - Check existing impressions/clicks for uncovered cities

### Process for Each City
1. Pull volume data for all 65 keyword patterns
2. Record: search volume, keyword difficulty, CPC, current ranking position (if any)
3. Identify keywords where TML already ranks on page 2-3 (quick-win optimization)
4. Flag high-volume / low-difficulty keywords for immediate content optimization
5. Note any city-specific keyword variations (e.g., "SEO Kitchener-Waterloo" vs "SEO Kitchener")

### Expected Output Per City
- Keyword spreadsheet with volume, KD, CPC, current rank
- Priority keywords for on-page optimization
- Content gap recommendations for location-service pages
- Local competitor list for each city
