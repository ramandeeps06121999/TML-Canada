## Executive Summary

TML targets high-competition keywords (KD 65-77) including "marketing agency," "seo company," "web design company," and city-specific variants like "seo company toronto" and "digital marketing agency toronto." Ranking for these requires a Domain Rating (DR) of 50+ with a strong backlink profile of relevant, authoritative Canadian links. This strategy targets 200+ quality backlinks in 6 months through directories, local citations, content marketing, PR outreach, and strategic partnerships.

## 6. Competitor Link Gap Analysis Framework

### Step 1: Identify Top Competitors

For each high-KD keyword, identify the top 5 ranking domains:

| Keyword | Expected Competitors (to research) |
|---------|-----------------------------------|
| seo company toronto | Major Player SEO, Let's Get Optimized, SEO.ca, Topspot, Consultus Digital |
| digital marketing agency toronto | Directive, Thrive, WebFX (if ranking in Canada), Major Tom, Awkward Media |
| web design company | Webflow agencies, Algorithmia, Numero Uno Web Solutions |
| marketing agency | Cossette, Dentsu Canada, Sid Lee |

### Step 2: Extract Competitor Backlink Profiles

Using Ahrefs, Semrush, or Moz:

1. **Enter each competitor domain** into the backlink analysis tool
2. **Export all referring domains** for the top 10 competitors
3. **Combine into a master spreadsheet** with columns:
   - Referring domain
   - DR of referring domain
   - Which competitor(s) have a link from this domain
   - Does TML have a link? (Yes/No)
   - Link type (editorial, directory, guest post, resource page, etc.)

### Step 3: Identify Link Gaps

Filter the master spreadsheet for:
- **Domains linking to 2+ competitors but NOT TML** (highest priority)
- **DR 40+ domains** linking to any competitor but not TML
- **Canadian domains (.ca)** linking to competitors but not TML
- **Industry-relevant domains** (marketing, business, tech blogs)

### Step 4: Categorize Gap Opportunities

| Category | Action |
|----------|--------|
| Directories where competitors are listed | Submit TML listing |
| Blogs where competitors guest-posted | Pitch guest post with better content |
| Resource pages listing competitor | Request inclusion via outreach email |
| News articles quoting competitors | Pitch TML as alternative source for future coverage |
| Broken links to competitors | Offer TML content as replacement (broken link building) |
| Sponsored/partner pages | Evaluate if same sponsorship is worthwhile |

### Step 5: Prioritize and Execute

Score each opportunity:
- **Relevance** (1-5): How relevant is the linking site to TML's services?
- **Authority** (1-5): What is the DR/DA of the linking domain?
- **Achievability** (1-5): How likely is TML to get this link?
- **Total Score** = Relevance + Authority + Achievability (max 15)

Execute in order of highest total score.

### Step 6: Recurring Gap Analysis

- Run this analysis **monthly** for the first 3 months, then **quarterly**
- Track new competitor links in Ahrefs Alerts
- Set up Ahrefs alerts for competitor new backlinks to catch opportunities in real time
