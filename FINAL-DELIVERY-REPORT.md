### 1. Organization Schema
```

Use on: Every page (global)
```

### 2. LocalBusiness Schema
```
Includes: Address, phone, hours, geo coordinates, reviews, map link
Shows: Google Maps, local pack, business cards
Use on: Service-location pages (e.g., seo-in-edmonton)
```

### 3. Service Schema
```

Use on: Every service page
```

```

Shows: Breadcrumb trail in search results
Use on: Every page
```

### 5. FAQPage Schema
```
Includes: FAQ questions with expanded answers

Use on: Pages with FAQ sections
```
